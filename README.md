Python Physics Simulation CollectionPygameとNumPyを使用して、古典力学の様々な現象を視覚化したシミュレーションライブラリです。物理定数、微分方程式の数値解法（オイラー法）、ベクトル演算の基礎をコードを通して学ぶことができます。🧪 収録されているシミュレーション1. 放物運動 (Projectile Motion)重力加速度 $g$ の影響下における物体の運動をシミュレートします。ベクトル演算による位置と速度の更新。画面外に出た際のリセットロジック。2. 弾性衝突 (Elastic Collision)2つのボールが衝突した際の速度変化を計算します。法線ベクトルと接線ベクトルを用いた速度成分の分解。運動量保存則に基づく速度の入れ替え。3. 多粒子系シミュレーション (Particle Swarm)100個の粒子が空間を自由に飛び回り、壁と跳ね返るデモンストレーションです。ランダムな初期値生成。効率的な多数オブジェクトの更新処理。4. 単振り子 & 複振り子 (Single & Double Pendulum)重力と張力による回転運動のシミュレーションです。単振り子: $\alpha = -(g/L) \sin \theta$ に基づく角加速度の計算。複振り子: 非線形な連立微分方程式を用いたカオス的挙動の再現。5. 単振動 (Simple Harmonic Motion)フックの法則（$F = -kx$）に基づくバネの運動です。復元力による加速度の変化。物体（質点）の振動エネルギーの可視化。🛠 技術スタックNumPy: ベクトルおよび行列演算の高速化。Pygame: リアルタイム描画とフレームレート（FPS）管理。オイラー法: 時間刻み幅 $dt$ を用いた数値積分による状態更新。🚀 使い方各スクリプトは独立して実行可能です。Bash# 依存ライブラリのインストール
